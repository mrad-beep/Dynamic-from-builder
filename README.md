# 📄 Dynamic Form Builder

A **Dynamic Form Builder** is a web application that allows users to create, customize, and manage forms without writing code. This project is built using **HTML, CSS, and JavaScript** for the frontend, focusing on simplicity, flexibility, and user interaction.

---

## 🚀 Features

* 🧩 **Dynamic Form Creation**
  Create forms with multiple field types such as:

  * Text Input
  * Dropdowns
  * Checkboxes
  * Radio Buttons

* 🎨 **Customizable UI**
  Modify form layout, styles, and structure dynamically.

* ➕ **Add / Remove Fields**
  Users can add new fields or delete existing ones in real time.

* 📝 **Live Preview**
  Instantly see how the form will look while building it.

* 📦 **Form Data Handling**
  Capture and process user input efficiently.

---

## 🛠️ Tech Stack

* **HTML5** – Structure of the application
* **CSS3** – Styling and responsive design
* **JavaScript (Vanilla JS)** – Dynamic behavior and DOM manipulation

## ⚙️ How It Works

1. User selects a field type (e.g., text, dropdown)
2. JavaScript dynamically creates the field
3. Field is appended to the form builder area
4. User can customize label, options, and structure
5. Final form can be used or submitted
